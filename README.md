#  Hospital Management System

A comprehensive **Hospital Management System desktop application** built
with **Python & PyQt5**.\
This project is designed to manage **patients, doctors, admissions**,
and provide a **powerful global search** across all records using
**SQLite**.

---

##  Features

####  Authentication
-   Simple login system to restrict access
-   Username & password based entry 

####  Patient Management
- Create, update, and delete patient records  
- Store essential medical data (vitals, history, medications)  
- View patient information in a structured table  

####  Doctor Management
- Register doctors with complete profiles  
- Manage specialty, department, and room assignment  
- Store contact information and work schedules  

####  Admission Management
- Manage patient admissions  
- Record case details, visit reasons, and insurance info  
- Track admission date and patient status  


####   Search System
-   Search across **all tables** (patients, doctors, admissions)
-   Keyword-based full database scan
-   Results displayed in a readable format
